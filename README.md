# Smart Cities & Resource Management Dashboard

An enterprise-grade, multi-page Power BI dashboard built to simulate a municipal management system. This platform monitors population movements, infrastructure health, and capital budget distributions across major urban districts.

## Key Features
* **Relational Data Modeling:** Connected multi-table Excel datasets tracking 300+ transaction migration logs and 150 capital portfolios.
* **Smart Diagnostic Alerts:** Custom DAX measures tracking grid strain with built-in conditional color shifts for threshold breaches (e.g., critical grid overloads over 80%).
* **Multi-Page Executive Navigation:** Seamless custom button navigation connecting operational health to deep financial project ledgers.
* **Nordic Slate UI:** A premium, high-density dark mode design layout optimizing scannability and reducing operator cognitive load.

## Tech Stack
* Power BI Desktop
* DAX (Data Analysis Expressions)
* Microsoft Excel (Data Generation & Interpolation)
